This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Dependencias

NextJS [Documentación](https://nextjs.org/docs) <br>
MantineUI [Components](https://mantine.dev/core/package/) <br>
MantineUI Form [Documentación](https://mantine.dev/form/package/) <br>
MantineUI Carousel [Documentación](https://mantine.dev/x/carousel/) <br>
MantineUI Dates [Documentación](https://mantine.dev/dates/getting-started/) <br>
MantineUI Dropzone [Documentación](https://mantine.dev/x/dropzone/) <br>
MantineUI Hooks [Documentación](https://mantine.dev/hooks/package/) <br>
MantineUI Modals [Documentación](https://mantine.dev/x/modals/) <br>
Mynaui Icons [Listado](https://mynaui.com/icons) <br>
DayJs [Documentación](https://day.js.org/docs/en/parse/now) <br>


Instalar dependencias:
```bash
npm i
```

## Desarrollo
Correr servidor de desarrollo:

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) para resultados en vivo.
